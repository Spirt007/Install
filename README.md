# Arch Install
INSTALLATION ON A BROKEN ALREADY DISK

ALL SECTIONS ARE FORMATTED EXT4

SIZE POINT YOURSELF

sda1 - root

sda2 - linux

sda3 - swap

sda4 - home

sdb5 - stock (second hard drive,not formatted)
